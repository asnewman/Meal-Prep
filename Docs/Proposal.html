Proposal:

Recipe and Meal prep web application where the user can find recipes based on ingredients. These recipes can be added to a calendar to plan when the meal is planned to be eaten. There will also be a notification panel that will notify on the day that the user has planned a meal. The user can also input all their available ingredients that the recipes will be generated from. The user can add and delete from this ingredients page.

Tools:

3rd Party API: Food2Fork http://food2fork.com/about/api

Features:

  • Search for recipes using specific ingredients
  • Input ingredients that you already have
  • Plan out meals ahead of time
  • Calendar scheduling in google calendar
  • Get notifications of planned meals
  • List of recently used ingredients
  • Like and dislike recipes
  • Comment on recipes
